JavaScript Bibliothek

  • Commenter(json_url)

    Erstellt eine neue Kommentarebene die sich über die gesamte Webseite legt und läd alle Kommentare aus der angegebenen URL.

    • hide()

      Versteckt die Kommentare.

    • show()

      Zeigt die Kommentare

    • addComment(author, content, time, [pos_x, pos_y])

      Fügt der Kommentarebene ein Kommentar hinzu. Fügt kein Kommentar in die Kommentar-Datei ein.

    • addCloseLink(commenter_object)

      Fügt einen Link zum Schließen der Kommentarebene ein.

    • userAddComment([pos_x, pos_y])

      Fügt ein Kommentar der Datei auf dem Server hinzu.

    • redrawLines()

      Zeichnet die Verbindungslinien zwischen den Kommentaren und den Kommentarpunkten neu
